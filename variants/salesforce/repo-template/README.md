## Quick start
1. Read `AGENTS.md`

5. Adjust package metadata and CI to your org model

## Repository areas
- `force-app/`: Salesforce metadata
- `manifest/`: package.xml and retrieval/deploy manifests
- `.agents/skills/`: reusable Codex workflows
- `.github/`: validation workflows and collaboration templates
- `docs/`: architecture, runbooks, and product context
