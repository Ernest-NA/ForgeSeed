# Goal
Build an LWC that is consistent, maintainable, and aligned with the UX and metadata model.

# Steps
1. Review the user flow and parent page context.
2. Reuse existing UI patterns where possible.
3. Keep controller logic clear and minimal.
4. Validate data access and error handling.
5. Add or update Jest tests if the project uses them.

# Constraints
- Avoid duplicating existing component patterns.
- Keep server interactions explicit and traceable.
